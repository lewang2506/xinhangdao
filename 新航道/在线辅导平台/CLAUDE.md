# 在线辅导平台 · 项目说明

## 重要同步规则

**`docs/UX答疑交互.md` 与 `docs/demo-ai-答疑-ipad.html` 必须保持一致。**

- 修改任意一侧（UX 文档或 HTML demo），必须同步更新另一侧。
- UX 文档是交互规范的单一来源，demo 是其可视化实现。
- 两者不一致时以 UX 文档为准。

## 设计风格规范（小清新）

### 色彩
| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色（柔和蓝） | `--blue` | `#5b8def` |
| 主色浅背景 | `--blue-lt` | `#eef3fd` |
| 辅色（柔和绿） | `--green` | `#6bbf8e` |
| 绿色浅背景 | `--green-lt` | `#eef8f2` |
| 青色 | `--teal` | `#5ab9b0` |
| 青色浅背景 | `--teal-lt` | `#eef8f7` |
| 警示红 | `--red` | `#e07070` |
| 背景灰 | `--gray-10` | `#f7f8fa` |
| 主文字 | `--text` | `#2c3340` |
| 辅助文字 | `--gray-60` | `#8e96a3` |

**禁止使用：** 高饱和度纯色、大面积深色渐变背景（如旧版 `#667eea → #764ba2`）。

### 风格原则
- 背景以浅灰白（`#f7f8fa`）为主，卡片用白色+细边框
- 强调色仅用于 icon、按钮、徽标等小面积元素
- hover 状态使用对应浅色背景 + 边框高亮，而非颜色加深
- 圆角普遍使用 12–16px，按钮 10–12px

## 页面流程（来自 UX 文档）

```
即时答疑首页
  ├─ AI 智能答疑
  │    └─ 拍照上传 → 题目确认 → AI 生成中 → 答案展示
  │                                               ├─ 下一题 → 拍照上传
  │                                               ├─ 呼叫在线老师 → [老师答疑流程]
  │                                               └─ 结束答疑 → 返回首页
  └─ 老师在线答疑
       └─ 二次确认弹框 → 选择学科 → 寻找老师（状态过滤 + 老师评分）
                                       → 积分确认弹窗（积分不足跳充值）
                                            → 等待老师接受（超时3分钟→返回列表）
                                                 → 直播间 → 直播结束页 → 返回首页
```

## 关键交互细节

- **积分制**：老师答疑消耗 30 积分/次（非现金）
- **弹窗动效**：`scaleIn` 缩放弹入
- **页面转场**：confirm→loading 全屏覆盖，loading→result 淡入，其余直接替换
- **结束行为**：AI 答疑结束直接返回首页；老师答疑（直播）结束进入直播结束页
- **直播结束页**：消耗积分 + 半星评分（1–5，支持0.5步长）+ 文字评价（非必填）→【回到首页】
- **横竖屏**：`.landscape` / `.portrait` class 切换，关键布局见 UX 文档横竖屏适配表
