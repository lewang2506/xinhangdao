# 老师后台 - UX 交互文档

## 整体架构

```
左侧导航栏                    右侧内容区
├── 用户中心                  ├── 老师信息概览
├── 直播方式设置              ├── 各功能模块详情页
├── 我的课程
│   ├── 课程日历
│   ├── 申请记录
├── 数据统计
└── 退出登录

管理者可见：
├── 老师管理
│   └── 申请审批
```

---

## 1. 老师用户中心

### 页面布局

**左侧：老师信息卡片**
- 头像（可点击更换）
- 老师姓名（只读，来自注册信息）
- 手机号（只读）

**右侧：信息编辑表单**

#### 基本信息区

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 头像 | 图片上传 | 是 | 建议尺寸 200x200px，支持 jpg/png |
| 真实姓名 | 文本 | 是 | 2-20字符，用于展示 |
| 手机号 | 文本 | 是 | 只读，注册时填写 |

#### 自我介绍区

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| 个人简介 | 多行文本 | 否 | 最多 80 字，用于老师卡片展示 |

#### 老师卡片预览

在表单下方实时展示老师卡片效果：

```
┌─────────────────────────────────────┐
│  ┌───┐  王明辉                       │
│  │ 王 │                             │
│  └───┘                             │
│  数学 · 导数、函数、立体几何等         │ ← 自我介绍展示位置（限制2行）
│      ⭐ 4.92                         │
│                                     │
└─────────────────────────────────────────────────────┘
```

**说明**：
- 个人简介控制在 80 字以内，展示在老师卡片中（见上图位置）
- 过长时截断显示"..."
- 实时预览确保老师知道展示效果

## 2. 直播方式设置

### 页面布局

页面分为两个独立设置卡片：

```
┌────────────────────────┬────────────────────────┐
│ 即时答疑设置            │ 预约指导设置            │
└────────────────────────┴────────────────────────┘
```

### 2.1 即时答疑设置

#### 参与开关
- 开关控件：【参与即时答疑】
- 关闭时整个卡片置灰不可编辑

#### 可授科目（多选）

即时答疑仅支持 A-Level 科目：

| 学科分类 | 可选科目 |
|----------|----------|
| 理科 | 数学、物理、化学、生物 |
| 商科 | 经济、会计、商务 |
| 文科 | 地理、历史、心理学 |

**展示方式**：分组复选框

```
理科
☑️ 数学   ☑️ 物理   ☐ 化学   ☐ 生物

商科
☑️ 经济   ☐ 会计   ☐ 商务

文科
☐ 地理   ☐ 历史   ☐ 心理学
```

#### 在线状态设置

**简化设计**：即时答疑采用在线/离线状态切换

```
┌─────────────────────────────────────────────────────────┐
│  在线状态                                                │
├─────────────────────────────────────────────────────────┤
│                                                         │
│   ○ 离线                           ● 在线              │
│                                                         │
│   当前状态：离线                                         │
│                                                         │
│   ┌─────────────────────────────────────────────────┐   │
│   │                                                 │   │
│   │        点击切换到在线状态，学生将可以看到         │   │
│   │        您并发起即时答疑请求                       │   │
│   │                                                 │   │
│   │              [切换为在线]                        │   │
│   │                                                 │   │
│   └─────────────────────────────────────────────────┘   │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**状态说明**：

| 状态 | 学生端表现 | 老师操作 |
|------|-----------|----------|
| 离线 | 老师不在老师列表中显示 | 可随时切换为在线 |
| 在线 | 老师在列表中显示，学生可发起请求 | 可随时切换为离线 |

**在线时的提醒**：
- 切换为在线时，显示提示"您已设置为在线状态，学生可以发起即时答疑请求"
- 页面顶部显示在线状态标识（绿色圆点 + "在线"）

**离线时的提醒**：
- 切换为离线时，确认弹窗"确定要设置为离线吗？当前进行中的答疑不会中断"
- 如有进行中的答疑，显示警告"您有进行中的答疑，结束后将自动切换为离线"

#### 电话提醒

**模块结构**

```
电话提醒 [在线/离线状态tag]
学生在线呼叫时，允许系统电话提醒我                          [允许/不允许下拉框]
```

**位置**：即时答疑设置卡片中，在线状态设置下方

**布局**：
- 左侧：电话图标 + "电话提醒"标题 + 状态tag（在线/离线）
- 中间：提示文字"学生在线呼叫时，允许系统电话提醒我"
- 右侧：下拉选择框（允许/不允许）

**状态对应关系**：

| 下拉选项 | 状态tag | 颜色 |
|----------|---------|------|
| 允许 | 在线 | 绿色 |
| 不允许 | 离线 | 红色 |

**交互说明**：

- 下拉切换时同步更新状态tag
- 切换到"允许"：显示绿色"在线"tag，toast提示"您已设置为允许电话提醒"
- 切换到"不允许"：显示红色"离线"tag，toast提示"您已设置为不允许电话提醒"

**默认值**：允许（在线状态）

**表格说明**：

| 单元格状态 | 显示 | 说明 |
|------------|------|------|

**已被预约时段锁定机制**：

| 场景 | 单元格状态 | 操作权限 |
|------|-----------|----------|
| 时段未被预约 | 可修改（蓝/白） | 可点击切换状态 |
| 时段已被预约 | 🔒 锁定（橙色） | **不可修改，点击提示"该时段已有预约"** |
| 取消申请审批中 | ⏳ 审核中（黄色） | **不可修改，提示"取消申请审核中"** |
| 取消已批准 | ❌ 已关闭（灰色） | **永久关闭，当日不可再开放** |

**单元格交互**：

| 操作 | 效果 |
|------|------|
| 单击单元格 | 切换状态：不可预约 → 可预约 → 不可预约 |
| 双击单元格 | 快速设置为"可预约" |
| 拖拽选择 | 批量选择多个单元格，统一设置状态 |
| 右键单元格 | 弹出菜单：复制 / 粘贴 / 清空 |



**快捷键**：

| 快捷键 | 功能 |
|--------|------|
| Ctrl + C | 复制选中的列（需在列头选择） |
| Ctrl + V | 粘贴到选中的列 |
| Delete | 清空选中的单元格 |
| 方向键 | 在单元格间移动 |
| Enter | 确认当前单元格选择并下移 |
| Space | 切换当前单元格状态 |



**法定节假日标注**：

- 节假日需要在列头标记出是什么节日


**时段规则**：
- 每个时段 1 小时（09:00-22:00，共13个时段）
- 支持一次显示 7 天（一周），可翻页查看

## 3. 我的课程

### 3.1 课程日历

**页面布局**：

```
┌─────────────────────────────────────────────────────────────┐
│ Tab: [预约指导]                    [今日] [周] [月] │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  [日期选择器]              [筛选：全部 | 待上课]            │
│                                                             │
│  ┌─────────────────────────────────────────────────────┐   │
│  │                                                     │   │
│  │              日历 / 列表 视图区域                      │   │
│  │                                                     │   │
│  └─────────────────────────────────────────────────────┘   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**视图模式**：

1. **日历视图（默认）**
   - 月视图展示
   - 有课程的日期显示标记点
   - 点击日期展开当日课程列表

2. **列表视图**
   - 按时间倒序排列
   - 展示课程基本信息

**课程卡片字段**：

| 字段 | 说明 |
|------|------|
| 课程类型 | 即时答疑 / 预约指导 |
| 时间 | YYYY-MM-DD HH:00 - HH:00 |
| 科目 | 数学、物理等 |
| 学生信息 | 学生昵称（脱敏）|
| 课程状态 | 待上课 |
| 操作 | 进入教室 / 申请取消 |

**说明**：课程日历仅显示"待上课"状态的课程，不显示已完成和已取消的课程。

### 3.2 申请记录

**入口**：在"待上课"状态的课程卡片上显示【申请取消】按钮

**申请弹窗**：

```
┌────────────────────────────────────────────────────────┐
│  申请取消课程                                           │
├────────────────────────────────────────────────────────┤
│                                                        │
│  课程信息：                                             │
│  ─────────────────────────────────────────             │
│  时间：2026-04-15 19:00 - 20:00                         │
│  科目：数学                                             │
│  类型：预约指导                                         │
│  ─────────────────────────────────────────             │
│                                                        │
│  取消原因 *（必填）                                      │
│  ┌──────────────────────────────────────┐              │
│  │                                      │              │
│  │  请说明取消原因，方便我们优化服务...    │              │
│  │                                      │              │
│  └──────────────────────────────────────┘              │
│  0 / 200                                               │
│                                                        │
│                                                        │
├────────────────────────────────────────────────────────┤
│          [取消]               [提交申请]                │
└────────────────────────────────────────────────────────┘
```

**取消规则说明**：
- 老师取消积分退还给学生
- **审批同意后**：
  - 预约日程取消，课程移至"已取消"列表
  - 该时段自动标记为"不可用"，不可再被预约
  - 系统自动退还学生积分并发送通知
- **审批拒绝后**：
  - 课程恢复正常"待上课"状态
  - 老师需按原计划上课
- 审批结果通过站内信和短信通知老师和学生

**取消后时段锁定机制**：

| 状态 | 时段显示 | 说明 |
|------|----------|------|
| 审批通过 | ❌ 不可用 | 该时段永久关闭，当日不可再开放 |
| 审批拒绝 | ● 可预约 | 恢复正常，可被预约 |

**申请记录页面**：

页面包含两个Tab：【审核中】和【申请记录】

**审核中表格字段**：

| 字段 | 说明 |
|------|------|
| 申请时间 | YYYY-MM-DD HH:MM |
| 课程时间 | MM-DD HH:00-HH:00 |
| 科目 | 数学/英语等 |
| 学生姓名 | 学生昵称 |
| 类型 | 取消申请类型 |
| 状态 | 审核中/已通过/已拒绝 |

**申请记录表格字段**：

| 字段 | 说明 |
|------|------|
| 审批时间 | YYYY-MM-DD HH:MM |
| 申请时间 | YYYY-MM-DD HH:MM |
| 课程时间 | MM-DD HH:00-HH:00 |
| 科目 | 数学/英语等 |
| 学生姓名 | 学生昵称 |
| 类型 | 取消申请类型 |
| 审批反馈 | 管理员审批时填写的反馈 |
| 审批结果 | 已通过/已拒绝 |

### 响应式断点

| 断点 | 宽度 | 布局调整 |
|------|------|----------|
| 大屏 | ≥1280px | 导航固定 200px，内容区自适应 |
| 中屏 | 992-1279px | 导航收缩为 64px 图标模式 |
| 小屏 | < 992px | 导航隐藏，通过汉堡菜单展开 |

### 通用组件

**按钮层级**：
- Primary：主操作（蓝底白字）
- Default：次要操作（白底灰边）
- Dashed：虚线按钮（用于添加）
- Text：文字按钮（用于编辑/删除）
- Link：链接样式

**表单规范**：
- Label 左对齐，必填项标记 *
- 输入框高度 32px
- 错误提示红色文字，出现在输入框下方
- 表单分组使用卡片分隔

**表格规范**：
- 表头灰色背景
- 行高 48px
- 操作列固定右侧
- 空数据状态显示图标+文字提示

**弹窗规范**：
- 确认类弹窗：宽度 400px
- 表单类弹窗：宽度 520px
- 详情类弹窗：宽度 640px
- 点击遮罩可关闭（配置项）
