# 即时答疑 - 学生端 UX 交互文档

## 完整用户流程

```
学生
  └─ 首页（批改界面）
        │
        ├─ 点击【名师在线】→ 老师列表页（带底部Tab）
        │
        ├─ AI答疑入口
        │    └─ 上传题目
        │         └─ AI识别展示答案
        │               ├─ 再问一题 ─→ 上传题目（循环）
        │               ├─ 呼叫在线老师 ─→ 老师列表页（直接跳转）
        │               └─ 结束答疑 → 返回首页
        │
        └─ 底部Tab导航（老师列表页、答疑管理页等）
              │
              ├─ ⚡ 即时答疑Tab
              │    ├─ 服务引导页 → 点击【立即呼叫老师】→ 老师列表
              │    └─ 老师列表页 → 积分确认 → 等待老师 → 直播间
              │
              └─ 📚 答疑管理Tab
                     ├─ 进行中：显示进行中的答疑 → 进入教室
                     └─ 已完成：历史答疑记录 → 观看回放
```

---

## 页面流程与交互

### 1. 即时答疑入口（首页）- 批改界面样式

**布局**
- 深灰色背景（#3a3f4a）
- 顶部标题栏：左侧返回按钮 + 中间题目序号 + 右上角"名师在线"圆形按钮
- 内容区：左右分栏布局

**左侧面板**
- 题目原题卡片：显示题目内容
- 用户答案卡片：显示用户的作答内容

**右侧面板（讲解区）**
- 顶部标签栏：答案 / 翻译 / 讲解（可切换）
- 下方内容区：展示对应讲解内容

**名师在线按钮**
- 位置：右上角
- 样式：圆形按钮，蓝色渐变背景，带图标
- 文字："名师在线"

**交互**
- 点击【名师在线】→ 直接进入老师列表页（跳过学科选择和二次确认）
- 点击【返回】→ 返回学习机首页

**说明**
- 首页（批改界面）**没有底部Tab导航**
- 底部Tab只在老师列表页、答疑管理页等二级页面出现

---

## 2. 底部Tab导航

在老师列表页、答疑管理页等二级页面显示底部Tab导航。

**Tab结构**
- ⚡ 即时答疑（原来的老师答疑流程）
- 📚 答疑管理

**样式**
- 位置：页面底部固定
- 高度：约60-70px
- 背景：白色/浅色
- 选中状态：图标和文字高亮显示

---

### 2.1 即时答疑Tab

**页面标题**
即时答疑

**内容**
- 即时答疑服务引导说明
- 【立即呼叫老师】主按钮

**交互**
- 点击【立即呼叫老师】→ 进入老师列表页

---

### 2.2 答疑管理Tab

**页面标题**
答疑管理

**顶部标签栏**
- 进行中
- 已完成

#### 进行中标签

**说明**
- 同时只能有一个进行中的答疑
- 显示当前正在进行的答疑卡片

**卡片内容**
- 科目标签（如：数学）
- 标题：科目 · 日期（如：数学 · 2024-01-18）
- 老师信息（头像 + 姓名）
- 【进入教室】按钮（蓝色主按钮）

**交互**
- 点击【进入教室】→ 进入直播间

#### 已完成标签

**说明**
- 显示历史答疑记录列表
- 按时间倒序排列（最新的在上）

**卡片内容**
- 科目标签
- 标题：科目 · 日期
- 老师信息 + 辅导时长
- 【观看回放】按钮

**交互**
- 点击【观看回放】→ 播放答疑回放视频

## 路径一：AI 答疑

### 2-A. 拍照上传题目

**布局**
- 顶部：【关闭】（左）
- 中部：相机取景框 + 对焦框线 + 提示文案"对准题目，点击拍照"
- 底部：快门按钮

**说明**
- 每次只拍一题，不支持多题连拍

**交互**
- 点击【关闭】→ 返回首页
- 点击快门 → 进入题目确认页

> 注：拍照页不提供"转老师答疑"入口，如需老师答疑请返回首页选择。

---

### 3-A. 题目确认页

**布局**
- 顶部：【返回】（左） + 标题"确认题目"（中）（右侧无按钮）
- 横屏：左侧题目图片，右侧 AI 识别结果
- 竖屏：图片在上，识别文字在下
- 底部：【重拍】+【开始解答】主按钮

**内容**
- 题目图片
- OCR 识别出的文字/公式

**交互**
- 点击【返回】→ 返回拍照页
- 点击【重拍】→ 返回拍照页
- 点击【开始解答】→ 进入 AI 生成中

---

### 4-A. AI 识别展示答案

**生成中状态**
- 居中 Loading 动效 + 文案"AI正在思考..."

**答案展示布局**
- 顶部：【返回首页】
- 横屏：左侧（1/3）原题图片 + 识别文字，右侧（2/3）分步解答
- 竖屏：原题在上，解答在下
- 底部操作栏：

```
[再问一题]   [呼叫在线老师]   [结束答疑]
```

**解答区内容**
- 分步骤说明（步骤序号 + 文字说明）
- 步骤间分隔线
- 底部：最终答案突出展示块

**交互**

| 操作 | 结果 |
|------|------|
| 点击【再问一题】 | 返回拍照页，继续上传新题目 |
| 点击【呼叫在线老师】 | 直接进入老师列表页（跳过学科选择） |
| 点击【结束答疑】 | 直接返回首页 |

---

## 路径二：老师答疑

> 触发入口：
> 1. 首页点击【名师在线】按钮
> 2. AI 答案页点击【呼叫在线老师】
> 3. 即时答疑Tab点击【立即呼叫老师】
>
> 以上入口均直接进入老师列表页，**跳过学科选择和二次确认弹窗**
>
> **注意**：老师列表页现在有底部Tab导航（⚡即时答疑 / 📚答疑管理）

### 2-B. 老师列表页

**布局**
- 顶部：【返回】（左） + 标题"在线老师"（中）
- 状态过滤标签栏：全部 / 空闲
- 老师列表区域

**状态过滤标签**
- 全部：显示所有状态的老师
- 空闲：仅显示状态为"空闲"的老师

**老师卡片字段**
- 头像
- 姓名
- 学科 + 擅长方向
- 学生评分
- 当前状态（空闲/忙碌）
- 右侧箭头

**老师状态说明**

| 状态 | 含义 |
|------|------|
| 空闲 | 老师飞书在线，但未在直播间中 |
| 忙碌 | 老师当前在直播间中 |

**学生评分展示规则**
- 无历史评分 → 显示"暂无评分"
- 有历史评分 → 显示平均分，保留两位小数（如 4.75）

**交互**
- 点击过滤标签 → 按状态筛选列表
- 点击空闲老师卡片 → 弹出积分确认弹窗
- 忙碌老师卡片不可点击（置灰显示）

---

### 3-B. 积分确认弹窗

| 项目 | 内容 |
|------|------|
| 邀请老师 | [老师姓名] |
| 辅导时长 | 30分钟（固定） |
| 消耗积分 | X积分 |
| 当前积分 | [用户当前积分] |

- 积分充足：底部显示【取消】 + 【确认呼叫】
- 积分不足：隐藏【确认呼叫】，显示提示"积分余额不足，请充值"，点击跳转充值页面（待确认）
- 点击【确认呼叫】→ 系统向老师发送飞书通知，进入等待老师接受页

---

### 4-B. 等待老师接受

**布局**
- 等待动效 + 已等待时长 + 文案"正在联系老师..."
- 显示已邀请的老师信息（头像 + 姓名 + 学科）

**等待结果**

| 结果 | 处理 |
|------|------|
| 老师接单并创建直播间 | 系统自动跳转直播页 |
| 超时（等待超过3分钟） | 弹框提示"您邀请的老师正忙，请选择其他老师"，点击【我知道了】返回老师选择列表 |

---

### 5-B. 直播间 & 视频答疑

**布局**
- 视频区域：老师视频（主画面）+ 学生视频（小窗）
- 题目区域：可展示学生上传的题目图片
- 底部工具栏：麦克风 / 摄像头 / 画板 / 结束

**交互**
- 点击【结束】或学生中途退出 → 进入直播结束页

---

### 6-B. 直播结束页

**布局**
- 提示文案"答疑结束"
- 本次消耗积分
- 评分区（非必填）：
  - 评分范围 1–5 分，支持半分（如 1.5）
  - 文字输入框：可输入文字评价
- 底部：【回到首页】按钮

**交互**
- 点击【回到首页】→ 返回学习机首页（退出答疑模块）

## 横竖屏适配（AI答疑部分）

| 页面 | 横屏布局 | 竖屏布局 |
|------|---------|---------|
| 首页（批改界面） | 左右分栏（题目区+讲解区） | 上下排列 |
| 确认页 | 图片/识别结果左右分栏 | 图片在上，识别结果在下 |
| 解答页 | 原题/解答左右分栏 | 原题/解答上下排列 |
| 拍照页 | 取景框宽扁（450×320） | 取景框方形（350×350） |

---

## 异常处理

| 异常场景 | 处理方式 |
|---------|---------|
| 拍照模糊/无法识别 | 提示"识别失败，请重新拍摄清晰照片" |
| 网络中断 | 提示"网络异常，点击重试" |
| AI生成超时 | 提示"AI思考超时，建议呼叫在线老师" |
| 积分不足 | 积分弹窗内提示"积分余额不足，请充值"，点击跳转充值页 |
| 无在线老师 | 列表为空时提示"当前暂无空闲老师，请稍后再试" |
| 老师超时未接受（>3分钟） | 弹框提示"您邀请的老师正忙，请选择其他老师"，点击【我知道了】返回老师列表 |

---

## 页面转场

- 首页 → 拍照：直接替换
- 拍照 → 确认：直接替换
- 确认 → 生成中：全屏 Loading 覆盖
- 生成中 → 解答展示：淡入
- 首页/解答页 → 老师列表页：直接替换（跳过学科选择）
- 等待老师 → 直播间：系统自动跳转（全屏切换）
- 直播间 → 直播结束页：直接替换
- 任意页 → 积分弹窗：弹窗缩放弹入（scaleIn 动效）
