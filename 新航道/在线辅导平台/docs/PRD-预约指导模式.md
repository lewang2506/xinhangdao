## 2. 项目背景

### 2.1 背景概述
新航道A level学员在大班课中，可能存在某个概念完全没掌握的情况。由于隐私顾虑（别人不知道自己成绩）和求助无门（大班老师无法为个人倒回去讲），需要有针对性的私密一对一辅导。预约指导模式让学生可以按自己节奏，预约特定老师进行系统性辅导。

### 2.2 目标用户

**核心用户：新航道A level培训学员**

| 角色细分 | 痛点（表层） | 痛点（人性本质） | 核心诉求 | 现有解决方案 | 机会点 |
|---------|-------------|----------------|---------|-------------|--------|
| **概念重建型**<br>（大班听课但没懂） | 某个概念完全没掌握 | **隐性落后的焦虑**——别人好像都懂了就我懵着；**求助无门的迷茫**——大班老师没法倒回去讲；**自我怀疑的羞耻**——是不是我笨才没懂；**累积崩塌的恐惧**——这个概念不懂，后面的课怎么办 | 1. **被单独看见**——有人专门为我停下来<br>2. **倒带式讲解**——从我不会的那个点开始<br>3. **私密安全**——一对一，没人知道我问了多么"基础"的问题<br>4. **一次彻底搞懂**——不是应付过去，是真的重建认知 | 1. 大班课硬着头皮听（越来越懵）<br>2. 自己查资料/看网课（不知从何补起）<br>3. 问同学（担心暴露+讲不系统）<br>4. 额外报小班（时间固定、成本高）<br>5. 攒到考试前集中补（问题越积越多） | 1. **私密一对一**——学习机内预约，没人知道你问了什么<br>2. **倒带式讲解**——老师从你卡住的那个点开始，不是从中间插进来<br>3. **灵活预约**——发现没懂立刻约，不等到问题堆积<br>4. **新航道师资**——懂A level课程体系，知道怎么补最有效 |
| **海外预备型** | 担心出国听不懂课 | **对未知的恐惧**——离开舒适区的不安全感；**完美主义**——不想在异国出丑 | 把"不确定的未来"变成"已准备好的现在" | 1. 自己找资料预习（无指导）<br>2. 国外网课（语言障碍）<br>3. 等出国后再适应（风险高） | 新航道特色海外课程辅导，以国内时间预约，中文讲解海外大学课程内容 |

## 3. 产品目标

### 3.1 核心目标
1. 打造"老师IP化"的预约辅导体系，让学生根据口碑选择
2. 提供灵活预约+课时套餐，降低尝试门槛
3. 针对海外预备学生，提供中文授课的海外课程辅导

### 3.2 成功指标

| 指标类型 | 具体指标 | 目标值 |
|---------|---------|--------|
| **服务质量** | 课程完课率 | ≥ 85% |
| | 用户满意度 | ≥ 4.5/5 |
| **用户活跃** | 月预约课程数 | ≥ 2次/人 |
| **转化** | 试用→购买课时包转化率 | ≥ 30% |
| | 即时解答→预约转化 | ≥ 15% |
| **留存** | 次月留存率 | ≥ 60% |
| | 课时包复购率 | ≥ 50% |
| **收入** | 单用户月均消费 | ≥ ¥300 |

## 4. 产品架构

### 4.1 服务模式

| 模式类型 | 核心定位 | 用户场景 | 时间特性 | 匹配机制 |
|---------|---------|---------|---------|---------|
| **预约上课** | 老师IP授课 | 提前安排学习计划<br>某个概念系统突破 | 灵活预约<br>可购1h/5h/10h课程包 | 选择在线老师直接呼叫<br>或预约特定时间 |
| **海外辅导** | 针对海外学生课程辅导 | 海外大学课程预习<br>经济学/数学等 | 以国内时间为准<br>灵活预约 | 选择海外认证老师预约 |

### 4.2 收费模式

| 服务类型 | 计费方式 | 价格策略 |
|---------|---------|---------|
| 预约上课 | 按课时收费 | 单小时定价，课时包优惠（买5送1等） |
| 海外辅导 | 待定 | 待确认是否与预约模式一致 |

### 5.1 学生端功能

#### 5.1.1 老师发现

**功能描述**：浏览和筛选可预约的老师

**核心功能点**：
- [ ] **老师列表**：展示可预约老师，支持列表/卡片视图
- [ ] **学科筛选**：按学科筛选（数学/物理/化学/经济等）
- [ ] **价格筛选**：按价格区间筛选
- [ ] **评分排序**：按评分、预约量、 newest 排序
- [ ] **老师搜索**：支持老师姓名/关键词搜索
- [ ] **推荐位**：首页展示热门/优质老师

#### 5.1.2 老师详情

**功能描述**：查看老师详细介绍和可约时间

**核心功能点**：
- [ ] **老师简介**：头像、昵称、教龄、擅长学科
- [ ] **IP标签**：如"概念讲解王者"、"A level数学专家"等
- [ ] **学生评价**：评分、评价数量、评价详情列表
- [ ] **历史课程数**：累计授课时长/学生数
- [ ] **可约日历**：展示近两周可预约时间段
- [ ] **课程价格**：单小时价格和课时包优惠
- [ ] **立即预约按钮**：进入预约流程

#### 5.1.3 预约下单

**功能描述**：选择时间并购买课时完成预约

**用户流程**：
```
选择老师 → 选择时间 → 选择课时包 → 确认支付 → 预约成功
```

**核心功能点**：
- [ ] **时间选择**：日历式时间选择器，显示可约/已约时段
- [ ] **课时包选择**：
  - 单次1小时
  - 套餐5小时（优惠X%）
  - 套餐10小时（优惠Y%）
- [ ] **问题描述**：填写本次辅导想解决的具体问题
- [ ] **支付确认**：显示订单详情和价格，确认支付
- [ ] **预约成功**：显示预约详情，添加日历提醒

#### 5.1.4 我的预约

**功能描述**：管理已预约的课程

**核心功能点**：
- [ ] **预约列表**：按时间展示待上/已完成/已取消的课程
- [ ] **上课入口**：课前10分钟显示进入教室按钮
- [ ] **取消预约**：开课前X小时可免费取消
- [ ] **改期申请**：申请改期，老师确认后生效
- [ ] **课程详情**：查看预约时间、老师信息、问题描述

#### 5.1.5 课程评价

**功能描述**：课后对老师和课程进行评价

**核心功能点**：
- [ ] **星级评分**：1-5星综合评分
- [ ] **维度评分**：讲解清晰度、耐心程度、针对性等
- [ ] **文字评价**：填写文字评价
- [ ] **匿名选项**：可选择匿名评价

### 5.2 海外辅导功能

**功能描述**：针对海外大学课程的专项辅导

**核心功能点**：
- [ ] **课程分类**：经济学、数学等海外课程分类展示
- [ ] **海外认证老师**：标识懂海外课程体系的认证老师
- [ ] **时区辅助**：国内时间为准，同时显示海外主要时区参考
- [ ] **课程内容说明**：该课程对应的海外大学课程内容说明

### 5.3 教师端功能

#### 5.3.1 个人中心

**核心功能点**：
- [ ] **资料编辑**：头像、昵称、简介、教龄
- [ ] **IP标签设置**：设置个人专长标签和教学特色
- [ ] **定价设置**：设置单小时价格和课时包优惠
- [ ] **可授课时间设置**：设置每周可授课时间段
- [ ] **个人主页预览**：预览学生端看到的效果

#### 5.3.2 预约管理

**核心功能点**：
- [ ] **预约确认/拒绝**：处理学生的预约请求
- [ ] **日程查看**：日历视图查看已确认的预约
- [ ] **上课入口**：课前进入教室
- [ ] **改期处理**：处理学生的改期申请

#### 5.3.3 收入管理

**核心功能点**：
- [ ] **收入统计**：今日/本周/本月/累计收入
- [ ] **课时统计**：已完成课时数
- [ ] **提现申请**：申请提现到银行卡
- [ ] **账单明细**：收入明细记录

### 5.4 系统功能

#### 5.4.1 课时管理

**核心功能点**：
- [ ] **课时包购买**：支持1h/5h/10h套餐购买
- [ ] **自动扣减**：预约成功后自动扣减课时
- [ ] **剩余课时提醒**：课时不足时提醒购买
- [ ] **课时有效期**：设置课时包有效期（如12个月）

#### 5.4.2 提醒通知

**核心功能点**：
- [ ] **预约成功通知**：推送+短信
- [ ] **课前提醒**：课前24小时、1小时提醒
- [ ] **上课入口通知**：课前10分钟推送上课入口
- [ ] **改期/取消通知**：状态变更时通知双方

## 6. 非功能需求

### 6.1 性能需求
- 页面加载时间 ≤ 3秒
- 日历数据加载 ≤ 2秒
- 预约冲突检测实时响应
- 支持 ≥ 500人同时在线上课

### 6.2 安全需求
- 课程视频录制留存
- 学生/教师身份实名认证
- 支付安全（符合金融安全标准）
- 数据加密存储与传输

### 6.3 兼容性需求
- 适配主流学习机分辨率
- 支持横屏/竖屏切换
- 适配不同Android版本
