# 预约指导 - 学生端 UX 交互文档

## 完整用户流程

```
学生
  └─ 预约指导入口
        ├─ 【开始预约】
        │    └─ Step 1. 选择科目
        │         └─ Step 2. 描述问题（必填文字 + 可拍照）
        │               └─ Step 3. 老师列表（按科目过滤，可排序/搜索）
        │                     └─ 老师详情页
        │                           └─ 【预约此老师】
        │                                 └─ Step 4. 选择上课时间（日历）
        │                                       └─ 确认积分弹窗
        │                                             ├─ 积分充足 → 预约成功页 → 返回入口
        │                                             └─ 积分不足 → 充值 → 返回弹窗
        │
        └─ 我的预约（Tab）
              └─ 预约列表（待上课 / 已完成 / 已取消）
                    └─ 预约卡片（含课程简述）
                          ├─ 待上课：【进入教室】（老师未入则置灰）
                          │         + 【取消预约】（满足条件时展示，含取消计数及剩余时间）
                          │     └─ 老师进入直播间后【进入教室】激活 → 直播间
                          │           └─ 结束课程 → 课程结束页（含评价，非必填）→ 返回入口
                          └─ 已完成 / 已取消：仅展示信息
```

---

## 页面流程与交互

### 1. 预约指导入口

**布局**
- 顶部：标题"预约指导" + 副标题"一对一，按你节奏，彻底搞懂"
- 中部：【开始预约】主按钮（蓝色，带图标）
- 底部 Tab：【预约上课】 【我的预约】

**交互**
- 点击【开始预约】或 Tab【预约上课】→ 进入 Step 1 选择科目
- 点击 Tab【我的预约】→ 切换到我的预约列表

---

## 路径一：预约下单流程

> 流程顺序：选择科目 → 描述问题 → 老师列表 → 老师详情 → 选择时间 → 确认积分
> 设计原则：先明确科目和问题，进入老师列表时已有上下文，时间冲突时科目和问题描述自动保留

### 2. 选择科目（Step 1 / 4）

**布局**
- 顶部：【取消】（左，退出整个预约流程返回入口）+ 步骤指示"第 1 步 / 共 4 步" + "选择科目"
- 科目网格（2 列）：数学 / 物理 / 化学 / 经济 / 英语 / 其他
- 每个科目卡片：图标 + 科目名
- 底部：【下一步】（选中科目后激活）

**交互**
- 点击科目卡片 → 选中态（`--blue` 边框 + `--blue-lt` 背景）
- 点击【下一步】→ 进入描述问题页

---

### 3. 描述问题（Step 2 / 4）

**布局**
- 顶部：【返回】（左）+ 步骤指示"第 2 步 / 共 4 步" + "描述问题"
- 说明文案："告诉老师你希望解决什么问题，老师将提前备课"
- 文本输入区：多行输入框（必填），占位文案"例如：导数大题综合题思路不清晰，总是卡在第二步"
- 字数计数：右下角实时显示 xx / 300
- 快捷标签（横向滚动）：概念不懂 / 解题思路 / 公式记忆 / 刷题技巧 / 考试技巧，点击追加到输入框
- 拍照上传区：
  - 标题"上传相关题目（选填）"
  - 最多上传 3 张图片，每张有删除按钮
  - 【拍照】按钮（仅拍照，无相册入口）
- 底部：【下一步】（文本不为空时激活）

**交互**
- 点击快捷标签 → 标签文字追加到输入框末尾，标签变为选中态（`--blue-lt` 背景）
- 超出 300 字 → 字数变红，禁止继续输入
- 点击【拍照】→ 调起相机，拍摄后显示缩略图
- 点击缩略图右上角 × → 删除该张
- 文本为空时点击【下一步】→ 输入框红色边框 + 提示"请描述问题后再继续"
- 文本非空 → 点击【下一步】→ 进入老师列表页

---

### 4. 老师列表（Step 3 / 4）

**布局**
- 顶部：【返回】（左）+ 步骤指示"第 3 步 / 共 4 步" + 已选科目名（中，如"数学老师"）
- 搜索栏：支持老师姓名 / 关键词搜索（需点击【搜索】按钮或键盘确认键触发，非实时过滤）
- 排序筛选（右上角）：综合推荐 / 评分最高 / 最多预约
- 老师列表：按当前科目过滤，卡片式，每行 2 列（横屏）/ 1 列（竖屏）

**老师卡片字段**

| 字段 | 说明 |
|------|------|
| 头像 | 老师头像 |
| 姓名 | 老师昵称 |
| IP标签 | 最多展示 2 个，如"A-level数学专家""概念讲解王者" |
| 擅长学科 | 学科徽标，最多 3 个 |
| 评分 | ★ 4.92（有评分时展示；无评分时显示"暂无评分"） |
| 已辅导学生数 | xxx 人次 |

**交互**
- 搜索框输入后点击【搜索】按钮或键盘确认键 → 返回搜索结果；未确认前不触发搜索
- 点击排序筛选 → 底部 sheet 弹出选项，选择后刷新列表
- 点击老师卡片（任意区域）→ 进入老师详情页

---

### 5. 老师详情页

**布局**
- 顶部：【返回】（左）+ 标题"老师详情"（中）
- 老师信息区：
  - 头像（大图）
  - 姓名 + IP标签列表
  - 擅长学科标签
  - 个人简介（默认折叠 3 行，点击展开）
- 评价区：
  - 综合评分（大字展示）+ 评分分布条形图
  - 最新评价列表（默认展示 3 条，可点击"查看全部"展开）
- 可约日历区：
  - 展示未来 14 天的可约时段
  - 每日按时段显示（上午 / 下午 / 晚上），可约时段高亮，已约/不可约置灰
- 底部固定操作栏：【预约此老师】（主按钮，蓝色）

**时段状态颜色**

| 状态 | 颜色 |
|------|------|
| 可预约 | `--blue-lt` 背景 + `--blue` 文字 |
| 已约满 | `--gray-10` 背景 + `--gray-60` 文字，不可点击 |
| 所选中 | `--blue` 背景 + 白色文字 |

**交互**
- 点击评价区"查看全部" → 展开完整评价列表
- 点击【预约此老师】→ 进入 Step 4 选择上课时间

---

### 6. 选择上课时间（Step 4 / 4）

**布局**
- 顶部：【返回】（左）+ 步骤指示"第 4 步 / 共 4 步" + "选择时间"
- 老师名片（小横幅）：头像 + 姓名 + 评分，提示当前为哪位老师选时间
- 日历选择器：月视图，可按月切换，已约满日期置灰
- 时段选择：选中日期后，下方展示当日可约时段列表（按上午 / 下午 / 晚上分组），每段时长 1 小时
- 积分说明：本次预约消耗 xx 积分（小字，灰色）
- 底部：【确认预约】（选中时段后激活）

**交互**
- 点击可用日期 → 日期高亮 + 展开时段列表
- 点击可用时段 → 时段选中态
- 点击置灰时段 → 无响应
- 点击【确认预约】→ 弹出确认积分弹窗

---

### 7. 确认积分弹窗

**弹窗样式**：居中弹出（`scaleIn` 动效）

**弹窗内容**

| 字段 | 内容 |
|------|------|
| 老师 | 头像 + 姓名 |
| 上课时间 | 2026-04-05（周日）10:00–11:00 |
| 课程时长 | 1 小时 |
| 科目 | 已选科目 |
| 问题描述 | 已填写内容（折叠超过 2 行） |
| 本次消耗 | xx 积分 |
| 当前积分 | [用户当前积分余额] |

**底部按钮**：【取消】（线框次要） + 【确认预约】（蓝色主要）

**积分不足分支**
- 弹窗内展示：积分余额不足，显示缺口积分量
- 【确认预约】替换为【去充值积分】→ 跳转充值页
- 充值成功后自动返回确认积分弹窗

**交互**
- 点击【取消】→ 关闭弹窗，保留已选信息
- 点击【确认预约】→ 全屏 Loading
- 预约成功 → 进入预约成功页
- 预约失败（时段冲突）→ 弹窗内提示"该时段已被预约，请重新选择"，点击【重新选择】返回时间选择页（科目和问题描述保留）

---

### 8. 预约成功页

**布局**
- 顶部：大图标（绿色勾）+ 标题"预约成功！"
- 预约信息卡片：
  - 老师头像 + 姓名
  - 科目 + 上课时间 + 课程时长（1 小时）
  - 已消耗积分
- 提醒说明：课前 24 小时和 10 分钟将收到上课提醒
- 底部：【查看我的预约】（主按钮，唯一操作）

**交互**
- 点击【查看我的预约】→ 跳转"我的预约"列表，默认展示"待上课"Tab

---

## 路径二：我的预约

### 9. 我的预约列表

**布局**
- 顶部：标题"我的预约"
- Tab 筛选：待上课 / 已完成 / 已取消
- 预约卡片列表

**预约卡片字段（含课程简述，无独立详情页）**

| 字段 | 说明 |
|------|------|
| 老师头像 | 圆形，带背景色 |
| 老师姓名 | — |
| 科目+知识点 | 格式为"科目·知识点1、知识点2、知识点3等"，最多显示3个知识点，超出显示"等" |
| 日期+时段 | 格式为"YYYY-MM-DD HH:00–HH:00" |
| 免费取消时间 | 格式为"免费取消时间：YYYY-MM-DD HH:00"（仅待上课状态） |
| 操作按钮 | 进入教室 + 取消预约 |

**取消条件提示（展示在卡片底部，灰色小字）**

"待上课"卡片展示免费取消时间：
- 格式："免费取消时间：YYYY-MM-DD HH:00"
- 说明：开课前24小时为免费取消截止时间

**状态与操作映射**

| 状态 | 操作按钮 | 说明 |
|------|---------|------|
| 待上课 | 【进入教室】+ 【取消预约】 | 进入教室根据直播间状态变色（未进入灰色，已进入蓝色），Demo中始终可点击 |
| 已完成 | 无操作按钮 | — |
| 已取消 | 无操作按钮 | — |

**【取消预约】显示条件**：距开课 > 24 小时，且本月已取消次数 < 3 次；任一不满足则隐藏。

**交互**
- 点击 Tab → 切换状态筛选
- 点击【进入教室】（激活态）→ 进入直播间
- 点击【取消预约】→ 弹出取消确认弹窗

---

### 10. 取消预约弹窗

**弹窗内容**
- 标题："确认取消预约？"
- 积分提示："如果免费取消，本次消耗的 **30 积分**将退回至账户。否则无论是否取消，将扣除30积分"
- 免费取消次数："已免费取消 x 次，本月剩余 x 次（每月限 3 次）"
- 取消规则：
  - 限额未满时："在 YYYY-MM-DD HH:00 前可免费取消"
  - 限额已满时："本月已免费取消3次，不可再免费取消"（红色警示）
- 底部按钮：【再想想】 + 【确认取消】（`--red` 警示色）

**交互**
- 点击【再想想】→ 关闭弹窗
- 点击【确认取消】→ 全屏 Loading → 取消成功，Toast 提示"预约已取消，积分已退回" → 刷新列表，卡片移至"已取消"Tab

---

## 路径三：上课流程

### 11. 直播间

**布局**（横屏优先）
- 主画面：老师视频（大）
- 小窗：学生视频（右下角，可拖动）
- 顶部信息栏：老师姓名 + 课程计时
- 题目展示区（可收起）：显示学生预约时填写的问题描述及上传图片
- 底部工具栏：麦克风开关 / 摄像头开关 / 画板 / 聊天 / 结束课程

**画板功能**
- 点击【画板】→ 全屏画板覆盖，老师学生均可书写
- 支持不同颜色笔 / 橡皮 / 清空

**交互**
- 点击【结束课程】→ 弹出确认弹窗"确定结束本次课程？"，确认后进入课程结束页

---

### 12. 课程结束页

**布局**
- 顶部：大图标（蓝色勾）+ 标题"课程结束"
- 本次课程信息：
  - 老师头像 + 姓名
  - 科目 / 上课时间 / 课程时长（1 小时）
- 评价区（非必填）：
  - 分区标题："对这次课满意吗？（选填）"
  - 综合评分：1–5 星，支持半星（0.5 步长），点击星星选择
  - 文字反馈输入框：占位文案"说说老师的亮点或建议（选填，最多 200 字）"
- 底部：【提交并回到首页】（若已填写任一评价项则文案为"提交评价并回到首页"，否则为"跳过，回到首页"）

**交互**
- 点击星星 → 星星选中，支持半星（点击星形左半=半星、右半=整星）
- 文字反馈超出 200 字 → 字数变红，禁止继续输入
- 点击底部按钮 → 若有填写则提交评价，无论有无评价均返回预约指导入口

## 横竖屏适配

| 页面 | 横屏布局 | 竖屏布局 |
|------|---------|---------|
| 选择科目 | 科目网格 3 列 | 科目网格 2 列 |
| 描述问题 | 文字区左（2/3），拍照上传右（1/3） | 文字区在上，拍照上传在下 |
| 老师列表 | 每行 2 个老师卡片 | 每行 1 个老师卡片（全宽） |
| 老师详情 | 左侧老师信息 + 右侧日历（各半） | 信息区在上，日历在下 |
| 时间选择 | 日历左侧，时段列表右侧 | 日历在上，时段列表在下 |
| 直播间 | 横屏主态（老师大画面，工具栏底部） | 竖屏次态（老师大画面上，工具栏下） |
| 我的预约列表 | 每行 2 个预约卡片 | 每行 1 个预约卡片（全宽） |

## 异常处理

| 异常场景 | 处理方式 |
|---------|---------|
| 预约时段已被他人抢占（冲突） | 确认积分弹窗内提示"该时段已被预约"，点击【重新选择时间】返回时间选择页，科目和问题描述保留 |
| 积分不足 | 确认积分弹窗内提示充值入口，充值后自动返回弹窗 |
| 问题描述为空时点击下一步 | 输入框红色边框 + 底部提示"请描述问题后再继续" |
| 老师取消预约 | 推送通知"老师因故取消本次预约，积分已退回"，卡片移至"已取消"Tab |
| 取消次数已达本月上限（3次） | 卡片不显示【取消预约】按钮，卡片底部提示"本月取消次数已达上限（3/3）" |
| 距开课不足 24 小时 | 卡片不显示【取消预约】按钮，卡片底部提示"已超过取消截止时间" |
| 网络断开（直播中） | Toast 提示"网络已断开，正在尝试重连"，3 次重连失败后提示手动重连按钮 |
| 直播超时/异常结束 | 进入课程结束页，顶部展示"课程异常结束"提示 + 客服入口 |
| 老师超时未进入直播间（超开课 10 分钟） | 卡片展示"老师迟到超 10 分钟"提示 + 客服入口，【取消预约】按钮临时恢复显示（不占用月取消次数） |
| 无可约时段（老师日历全满） | 老师详情页提示"该老师近期已约满，可返回选择其他老师"，【预约此老师】按钮置灰 |
| 老师列表为空（该科目暂无老师） | 老师列表页展示空态插图 + "暂无该科目老师，请稍后再试" |

## 页面转场

| 来源 → 目标 | 转场方式 |
|------------|---------|
| 入口 → 选择科目 | 右侧滑入（push） |
| 选择科目 → 描述问题 | 右侧滑入（push） |
| 描述问题 → 老师列表 | 右侧滑入（push） |
| 老师列表 → 老师详情 | 右侧滑入（push） |
| 老师详情 → 选择时间 | 右侧滑入（push） |
| 任意页 → 确认积分弹窗 / 取消弹窗 | 弹窗缩放弹入（`scaleIn` 动效） |
| 确认积分 → 全屏 Loading | 全屏覆盖 |
| 全屏 Loading → 预约成功页 | 淡入 |
| 我的预约卡片【进入教室】→ 直播间 | 全屏切换 |
| 直播间 → 课程结束页 | 全屏切换 |
| 课程结束页 → 入口 | 左侧滑出（pop to root） |
