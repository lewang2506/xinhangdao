# 产品需求文档 (PRD)

## 在线辅导平台 - 学习机应用

---

## 1. 文档信息

| 项目 | 内容 |
|------|------|
| 产品名称 | 新航道在线辅导平台 |
| 产品形态 | 学习机应用（类似iPad） |
| 文档版本 | v1.0 |
| 创建日期 | 2026-03-18 |
| 文档状态 | 初稿 |

---

## 2. 项目背景

### 2.1 背景概述
基于学习机平台开发在线辅导功能，通过技术手段连接学生与优质教师资源，提供一对一服务和个性化辅导，满足不同场景下的学习需求。

### 2.2 目标用户
- **学生用户**：需要课业辅导、答疑解惑的中小学生
- **海外学生**：需要中文授课的海外课程辅导学生
- **教师用户**：提供辅导服务的在线教师

---

## 3. 产品目标

### 3.1 核心目标
1. 打造即时响应的在线答疑服务，解决学生学习中的即时困难
2. 建立预约上课体系，支持长期学习计划制定
3. 拓展海外辅导市场，服务海外中文学习者

### 3.2 成功指标
| 指标 | 目标值 |
|------|--------|
| 即时解答匹配成功率 | ≥ 90% |
| 平均匹配等待时间 | ≤ 5分钟 |
| 用户满意度 | ≥ 4.5/5 |
| 课程完课率 | ≥ 85% |

---

## 4. 产品架构

### 4.1 三种服务模式

| 模式类型 | 核心定位 | 用户场景 | 时间特性 | 匹配机制 |
|---------|---------|---------|---------|---------|
| **即时解答模式** | 类似滴滴/在线问诊 | 刷题中遇问题需即时解决 | 20-30分钟/次<br>5-10分钟内匹配 | 一键呼叫<br>系统自动匹配 |
| **预约上课模式** | 老师IP授课 | 提前安排学习计划<br>提分辅导 | 灵活预约<br>可购多小时课程 | 选择在线老师直接呼叫<br>或预约特定时间 |
| **海外辅导模式** | 针对海外学生课程辅导 | 海外大学课程辅导<br>如经济学/数学 | 以国内时间为准<br>灵活预约 | 待确认 |

### 4.2 即时解答子模式

| 子模式 | 服务形式 | 收费方式 | 适用场景 |
|-------|---------|---------|---------|
| **文字说明** | AI提供智能解答 | 按次收费（低价） | 基础问题、概念解释 |
| **在线解答** | 真人老师视频/语音答疑 | 按次+时长收费 | 复杂问题、需要互动讲解 |

---

## 5. 功能需求

### 5.1 即时解答模式

#### 5.1.1 功能描述
学生在学习过程中遇到问题时，可一键发起求助请求，系统快速匹配合适的教师进行在线解答。

#### 5.1.2 用户流程
```
学生发起请求 → 选择问题类型/学科 → 系统匹配老师 → 开始答疑 → 结束答疑 → 评价
```

#### 5.1.3 核心功能点

**学生端：**
- [ ] 一键呼叫功能（快速发起求助）
- [ ] 问题描述输入（文字/拍照上传题目）
- [ ] AI文字解答优先推荐
- [ ] 在线解答排队等待界面
- [ ] 视频/语音通话功能
- [ ] 白板/屏幕共享功能
- [ ] 答疑结束确认与评价

**教师端：**
- [ ] 在线状态切换（上线/离线）
- [ ] 接单抢单功能
- [ ] 通话与协作工具
- [ ] 课后问题归档

**系统端：**
- [ ] 智能匹配算法（基于学科、难度、教师专长）
- [ ] 排队等待机制
- [ ] 超时自动转派
- [ ] 服务质量监控

### 5.2 预约上课模式

#### 5.2.1 功能描述
学生可提前预约特定教师，按计划进行系统性的学习辅导。

#### 5.2.2 用户流程
```
浏览老师列表 → 查看老师详情/评价 → 选择时间 → 确认预约 → 支付 → 上课提醒 → 上课 → 评价
```

#### 5.2.3 核心功能点

**学生端：**
- [ ] 教师列表浏览（支持筛选：学科、评分、价格等）
- [ ] 教师详情页（简介、专长、评价、可预约时间）
- [ ] 日历式时间选择器
- [ ] 课程包购买（1小时/5小时/10小时套餐）
- [ ] 预约管理与提醒
- [ ] 取消/改期功能

**教师端：**
- [ ] 个人主页管理
- [ ] 可授课时间设置
- [ ] 预约确认/拒绝
- [ ] 课程记录管理

**系统端：**
- [ ] 课时扣减与结算
- [ ] 预约冲突检测
- [ ] 课前提醒推送（短信/推送）

### 5.3 海外辅导模式

#### 5.3.1 功能描述
专为海外学生设计的课程辅导服务，以中文授课，针对海外大学课程内容。

#### 5.3.2 特殊说明
- **时区处理**：以国内时间为准进行预约
- **语言**：第1期为中文授课
- **课程范围**：经济学、数学（待确认扩展学科）
- **收费策略**：待确认

#### 5.3.3 核心功能点
- [ ] 海外课程分类展示
- [ ] 时区显示转换（辅助海外学生理解）
- [ ] 专项教师标签（海外课程认证）

## 6. 商业模式

### 6.1 收费模式

| 服务类型 | 计费方式 | 说明 |
|---------|---------|------|
| AI文字解答 | 按次收费 | 低价策略，快速响应基础问题 |
| 即时在线解答 | 按次+时长收费 | 基础费用+按分钟计费 |
| 预约课程 | 按课时收费 | 套餐优惠（买多送多） |
| 海外辅导 | 待定 | 待确认定价策略 |

### 6.2 结算机制
- 学生预付费用到平台
- 服务完成后结算给教师
- 平台抽取服务佣金

---

## 7. 非功能需求

### 7.1 性能需求
- 页面加载时间 ≤ 3秒
- 视频通话接通时间 ≤ 5秒
- 系统并发支持 ≥ 1000人同时在线

### 7.2 安全需求
- 用户实名认证
- 通话内容合规监控
- 支付安全（符合金融安全标准）
- 数据加密存储与传输

### 7.3 兼容性需求
- 适配主流学习机分辨率
- 支持横屏/竖屏切换
- 适配不同Android版本
